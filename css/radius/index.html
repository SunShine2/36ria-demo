css3圆角Border-radius详解

demo制作：明河共影，欢迎访问我的博客。

文章链接：http://www.36ria.com/3087

这是第一个圆角示例，兼容 Firefox, Safari/Chrome, Opera, IE9

#demo-1{
border-radius:10px;
-moz-border-radius: 10px;
-webkit-border-radius: 10px;
}

这是第二个圆角示例，演示border-bottom-left-radius, border-bottom-right-radius, border-top-left-radius, border-top-right-radius的使用，这里以border-top-left-radius为例，留意左上角的圆角和火狐下的特殊写法

#demo-2{
border-top-left-radius: 100px 10px;
-moz-border-radius-topleft: 100px 10px;
}

W3C标准圆角写法和Mozilla圆角写法对照

W3C 的圆角属性  Mozilla （火狐）的圆角属性 
border-radius -moz-border-radius
border-top-left-radius -moz-border-radius-topleft
border-top-right-radius -moz-border-radius-topright
border-bottom-right-radius -moz-border-radius-bottomright
border-bottom-left-radius -moz-border-radius-bottomleft

这是第三个示例，演示如何使用圆角，制作一个圆，留意这里宽度、高度和圆角属性值的关系

#demo-3{
height: 70px;
width: 70px;
-moz-border-radius: 35px;
border-radius: 35px;
background-color:#999;
}
