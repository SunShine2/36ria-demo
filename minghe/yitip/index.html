jquery.yitip V0.1——jquery文字提示插件

作者：明河共影，博客：RIA之家

文章链接：http://www.36ria.com/2765

默认提示框

给所有带有title的目标容器加上提示框

$('[title]').yitip();

demo1：这是第一个标题

给目标容器加上color属性，即可更改提示框的颜色

目前共有以下颜色：黄色（默认）、蓝色、绿色、红色、黑色、白色

demo2：这是第二个标题

将提示框的位置改成目标容器之下

var demo2Tip = $("#demo2-title").yitip("api");
demo2Tip.setPosition("bottomMiddle");

目前你可以设置的位置为：上中、下中、左中、右中

demo3：这是第三个标题

给位置加上偏移值，这里是在默认的基础上向右移动10px

var demo3Tip = $("#demo3-title").yitip("api");
demo3Tip.options.offest = {"left":10,"top":0};
demo3Tip.setPosition("rightMiddle");

demo4：这是第四个标题

重新改变大标题提示框的内容、颜色、位置，代码如下：

var singleTitleTip = $("#single-title").yitip("api");
singleTitleTip.setContent('<img src="demo-images/1.png" width="159" height="135" />');
singleTitleTip.setPosition("rightMiddle");
singleTitleTip.setColor("blue");

demo5：这是第五个标题

ajax方式获取数据并修改提示框的内容

var demo5Tip = $("#demo5-title").yitip("api");
demo5Tip.setContent({url:"demo-html/test.html",type:"get"});

demo6：这是第6个标题

单击才出现提示框，手动关闭提示框

$("#demo6-title").yitip({"showEvent" :"click",
"hideEvent" : null});
var demo6Tip = $("#demo6-title").yitip("api");
demo6Tip.setContent({url:"demo-html/test2.html",type:"get",
success:function(data){
demo6Tip.$yitip.find("#close").click(function(){
demo6Tip.hide(0);
})
}});
